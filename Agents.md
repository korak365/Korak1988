# Hotel Rate Parity Checker

This tool checks for rate parity issues by comparing hotel prices between Expedia and the hotel's direct booking website.

## Features:
- Scrapes pricing data from Expedia and direct booking websites.
- Compares results and flags discrepancies in pricing.
- Provides data on price differences, URLs, and discrepancy flags for further analysis.

## Usage:
1. Input the same number of Expedia and direct booking URLs.
2. Run the scraper.
3. Review the dataset for flagged discrepancies.
